# Portfólio – Atividade da Faculdade

Landing page simples e moderna para apresentar uma atividade acadêmica: contexto, objetivos, tecnologias, resultados e links.

## Estrutura

- `index.html`: página principal com seções (Sobre, Objetivos, Tecnologias, Resultados, Galeria, Links).
- `styles.css`: estilos com tema escuro/claro, layout responsivo e animações leves.
- `script.js`: alternância de tema, menu mobile, smooth scroll e animações de revelação.

## Como executar

Você pode simplesmente abrir o `index.html` no navegador ou subir um servidor local:

### Python (se instalado)

```bash
cd "/c/Users/Leonardo/portifolioAVA"
python -m http.server 5500
```

Acesse: http://localhost:5500

### Node (opcional)

```bash
cd "/c/Users/Leonardo/portifolioAVA"
npx serve . -p 5500
```

## Personalização rápida

- Troque os textos das seções em `index.html` conforme sua atividade.
- Atualize os links de GitHub/Relatório/Slides na seção `Links`.
- Substitua os blocos da `Galeria` por imagens reais.
- Ajuste paleta de cores em `styles.css` (variáveis `:root`).
